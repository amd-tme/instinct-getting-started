# Nscale

## Overview

Nscale is a vertically integrated GPU cloud provider specializing in sustainable, high-performance AI computing. Spun out from Arkon Energy, a 300MW data center and hosting business, Nscale operates from Northern Norway, taking advantage of 100% renewable energy and natural cooling solutions to deliver cost-effective AMD Instinct GPU services.

## Available AMD Instinct GPU Instances

### AMD Instinct MI300X

Nscale offers AMD Instinct MI300X GPUs as part of their sustainable AI cloud platform:

**Key Specifications:**

- **GPU**: AMD Instinct MI300X accelerators
- **Memory**: 192GB of HBM3 memory per GPU
- **Performance**: Optimized for compute-intensive workloads
- **Environment**: 100% renewable energy powered
- **Cooling**: Natural cooling solutions leveraging Arctic location

### AMD Instinct MI250X

Also available through Nscale's platform:

**Features:**

- **GPU**: AMD Instinct MI250X accelerators
- **Use Cases**: AI training, inference, and HPC workloads
- **Integration**: Full ROCm software stack support

## Getting Started with Nscale

### Prerequisites

- Contact Nscale for account setup and capacity allocation
- Understanding of AI/ML workload requirements
- Familiarity with containerized deployments (Kubernetes/Slurm)

### Deployment Process

1. **Request Access**
   Visit [nscale.com](https://www.nscale.com) and apply for access to their AMD Instinct GPU platform.

2. **Choose Your Deployment Model**
   - **On-demand GPU compute**: For flexible, scalable workloads
   - **Reserved clusters**: For large-scale, dedicated GPU clusters
   - **API endpoints**: For instant and scalable Generative AI inference
   - **Bare metal resources**: For maximum performance requirements

3. **Configure Your Environment**
   - Select GPU count and configuration
   - Choose between Kubernetes and Slurm cluster management
   - Configure storage and networking requirements
