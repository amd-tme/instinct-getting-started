# IBM Cloud

## Overview

IBM Cloud is an enterprise-grade cloud computing platform designed for security, compliance, and hybrid cloud capabilities. IBM has announced a strategic collaboration with AMD to deploy AMD Instinct MI300X accelerators as a service on IBM Cloud, expanding their AI accelerator offerings to meet the growing demand for high-performance computing in AI workloads.

## Upcoming AMD Instinct GPU Instances

> **Note**: As of May 2025, IBM Cloud AMD Instinct GPU offerings are not yet generally available but are planned for release in the first half of 2025.

### IBM Cloud Virtual Servers for VPC with AMD Instinct MI300X

IBM has announced plans to offer AMD Instinct MI300X accelerators as a service on IBM Cloud Virtual Servers for Virtual Private Cloud (VPC).

**Expected Specifications:**

- **GPU**: AMD Instinct MI300X accelerators
- **Memory**: 192GB of HBM3 memory per GPU
- **Performance**: Optimized for large model inferencing and fine-tuning
- **Platform Integration**: Support for IBM's watsonx AI platform

### Container Support

IBM plans to provide container support for AMD Instinct MI300X accelerators through:

- IBM Cloud Kubernetes Service
- IBM Red Hat OpenShift on IBM Cloud

This approach aims to optimize performance for enterprises running AI applications in containerized environments.

## Anticipated Use Cases

### Enterprise AI Inferencing

The large memory capacity of AMD Instinct MI300X accelerators (192GB per GPU) will allow companies to:

- Run larger AI models with fewer GPUs
- Potentially lower costs for inferencing workloads
- Deploy memory-intensive applications more efficiently

### Integration with IBM watsonx

IBM plans to offer support for AMD Instinct MI300X accelerators within IBM's watsonx AI platform, providing:

- Additional AI infrastructure resources for watsonx customers
- Support for Red Hat Enterprise Linux AI inferencing
- Efficient GenAI inferencing workloads
