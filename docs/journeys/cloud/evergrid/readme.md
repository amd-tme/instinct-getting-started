# Evergrid AI

## Overview

Evergrid AI is a specialized AI infrastructure provider — the "inference neocloud" — focused on delivering high-performance, single-tenant AI clusters for labs, hyperscalers, enterprises, and research institutions. Evergrid designs, deploys, and operates large-scale AI infrastructure built on emerging hardware platforms, including AMD Instinct accelerators.

Evergrid participates in the AMD Instinct GPU Evaluation Program. Visit the [AMD Evaluation Program](https://www.amd.com/en/products/accelerators/instinct/eval-request.html) to apply for access.

## Services

Evergrid offers three primary service tiers:

### Managed AI

Application deployment and optimization services for production AI workloads — including dedicated inference endpoints with hardware-specific optimizations, multi-model routing, and enterprise-ready agent environments. Available with GPU-based or token-based pricing.

### Compute Platform

Managed Kubernetes, Slurm, and virtual machine environments backed by:

- High-performance CPUs
- Secure, low-latency networking
- Scalable persistent storage

Suitable for training, inference, and HPC workloads requiring flexible orchestration.

### Fleet & Infrastructure

Large-scale fleet operations with automated provisioning, health checks, and self-healing capabilities. Designed for deployment into gigascale AI factories with:

- Advanced cooling infrastructure
- End-to-end facility telemetry
- GPU-native redundancy

## Key Differentiators

- **Over 1 GW of capacity under development** — built to meet enterprise-scale requirements
- **24/7 monitoring with 12-minute response SLA** — dedicated engineering point of contact

## Getting Started with Evergrid AI

### Engagement Process

Evergrid is designed for enterprise and research engagements with dedicated infrastructure needs.

1. **Contact an Engineer**
   Visit [evergrid.ai](https://www.evergrid.ai/contact/) and speak directly with an Evergrid engineer to describe your requirements.

2. **Capacity Planning**
   - Discuss scale (number of GPUs, cluster topology)
   - Review hardware availability and deployment timeline
   - Define networking, storage, and orchestration requirements

3. **Cluster Deployment**
   Evergrid deploys and operates the infrastructure — including burn-in testing, health verification, and ongoing fleet monitoring.

4. **Workload Onboarding**
   Evergrid provides assistance with ROCm stack configuration, model deployment, and performance tuning.
