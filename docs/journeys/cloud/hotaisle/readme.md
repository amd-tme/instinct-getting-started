# Hot Aisle

## Overview

Hot Aisle is a specialized provider of high-performance bare metal compute solutions focused exclusively on AMD Instinct accelerators. Founded by industry veterans with extensive experience in open-source computing, large-scale deployments, and data center management, Hot Aisle delivers cutting-edge cluster solutions and consulting services for AI and HPC workloads.

## Available AMD Instinct GPU Solutions

### AMD Instinct MI300X Bare Metal Clusters

Hot Aisle specializes in AMD Instinct MI300X-based bare metal solutions:

**Key Specifications:**

- **GPU**: AMD Instinct MI300X discrete accelerators
- **Architecture**: Based on AMD CDNA 3 architecture
- **Memory**: 192GB of HBM3 memory per GPU
- **Compute Units**: 304 high-throughput compute units per GPU
- **Performance**: 13.7x peak AI/ML performance improvement over MI250X using FP8 with sparsity
- **HPC Performance**: 3.4x peak advantage for HPC workloads on FP32 calculations

### Flagship Cluster Configuration

**128 GPU MI300X Bare Metal Cluster:**

- **Total GPUs**: 128x AMD Instinct MI300X accelerators
- **Performance**: Designed to match or exceed H100 setup performance
- **Interconnect**: AMD Infinity Fabric technology with 128 GB/s bidirectional bandwidth between GPUs
- **Deployment**: Optimized for maximum performance and efficiency
