# TensorWave

## Overview

TensorWave is a next-generation cloud platform exclusively focused on AMD GPU infrastructure for AI workloads. Founded in 2023 and based in Las Vegas, TensorWave has positioned itself as the leading AMD-powered cloud provider, offering high-performance computing solutions specifically designed for AI training, fine-tuning, and inference.

## Available AMD Instinct GPU Instances

### AMD Instinct MI300X

TensorWave's flagship offering centers on AMD Instinct MI300X accelerators:

**Key Specifications:**

- **GPU**: AMD Instinct MI300X accelerators (8-GPU configurations)
- **Memory**: 192GB of HBM3 memory per GPU
- **Performance**: Optimized for memory-intensive demands of LLMs and deep learning
- **Platform**: Built specifically for AI workloads with purpose-built inference engine
- **Deployment**: Bare metal configurations for maximum performance

### AMD Instinct MI325X

TensorWave is deploying AMD Instinct MI325X GPUs as part of their expansion:

**Specifications:**

- **GPU**: AMD Instinct MI325X accelerators
- **Memory**: 256GB of HBM3E memory per GPU
- **Bandwidth**: 8 TB/s memory bandwidth
- **Cluster Size**: Building an 8,192 Instinct MI325X GPU cluster

### Future Expansion: AMD Instinct MI350X Series

TensorWave has announced plans to deploy AMD's next-generation MI350X series as they become available, aiming to build one of the world's largest AMD GPU clusters.

## Getting Started with TensorWave

### Prerequisites

- Contact TensorWave for account setup and access
- Understanding of AI/ML workload requirements
- Familiarity with AMD ROCm platform

### Deployment Process

1. **Request Access**
   Visit [tensorwave.com](https://tensorwave.com) and contact their team for access to AMD Instinct GPUs.

2. **Configure Your Workload**
   - Specify your AI model requirements
   - Choose between inference and training configurations
   - Select GPU count and duration

3. **Deploy and Scale**
   - Access your dedicated AMD Instinct GPU resources
   - Use TensorWave's optimized inference engine
   - Monitor performance through real-time GPU monitoring tools
