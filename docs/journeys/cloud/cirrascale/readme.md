# Cirrascale

## Overview

Cirrascale Cloud Services is a leading provider of innovative cloud and managed solutions for AI and high-performance computing (HPC). The company specializes in the AI Innovation Cloud, which is purpose-built to power the most demanding AI and HPC workloads using AMD Instinct accelerators.

## Available AMD Instinct GPU Instances

### AMD Instinct MI250

Currently available through the Cirrascale AI Innovation Cloud:

**Specifications:**

- **GPU**: 4x AMD Instinct MI250 accelerators per instance
- **CPU**: Dual 64-core processors
- **System Memory**: 1TB
- **Local Storage**: 960GB NVMe + 3.84TB NVMe
- **Network**: 25Gb bonded connection
- **Performance**: Up to 47.9 TFLOPS peak FP64 performance for HPC workloads

### AMD Instinct MI300X

**Specifications:**

- **GPU**: 8x AMD Instinct MI300X accelerators per instance
- **CPU**: Dual 48-core processors
- **System Memory**: 2.3TB
- **Local Storage**: 960GB NVMe + (4) 3.84TB NVMe drives
- **Network**: 25Gb bonded connection (3200Gb available)
- **GPU Memory**: 192GB HBM3 per GPU (1.5TB total)

### AMD Instinct MI325X

Now available in the Cirrascale AI Innovation Cloud:

**Specifications:**

- **GPU**: AMD Instinct MI325X accelerators
- **Memory**: 256 GB HBM3E per GPU with 8 TB/s memory bandwidth
- **Architecture**: 3rd Gen AMD CDNA (CDNA 3)
- **Features**: Superior FP16, BF16, FP8, and INT8 processing for leading GenAI models

### Upcoming: AMD Instinct MI350 Series

Cirrascale has announced the upcoming availability of:

- **AMD Instinct MI350 Series**: Built on 4th generation AMD CDNA architecture with up to 288 GB HBM3E memory

## Getting Started with Cirrascale AMD Instinct GPUs

### Prerequisites

- Contact Cirrascale for account setup and capacity allocation
- Understanding of AI/ML or HPC workload requirements
- Basic knowledge of AMD ROCm platform

### Deployment Process

1. **Contact Cirrascale Sales**
   Visit [cirrascale.com](https://www.cirrascale.com) and contact their sales team to discuss your requirements.

2. **Configure Your Environment**
   - Specify your compute, memory, and storage requirements
   - Choose the commitment term (monthly, 3-month, 6-month, or annual)
   - Configure networking and security settings

3. **Access Your Instance**
   - Receive connection details and credentials
   - Connect to your dedicated AMD Instinct GPU cluster
   - Verify ROCm installation and GPU availability
